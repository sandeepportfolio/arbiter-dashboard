<objective>
Restructure graceful shutdown in `arbiter/main.py` so that order cancellation happens BEFORE task cancellation, implement full `cancel_all()` on Kalshi (batched with chunking) and Polymarket (SDK cancel_all), and add `prepare_shutdown()` on SafetySupervisor that broadcasts a `shutdown_state` WebSocket event BEFORE the cancel fanout so the dashboard knows the server is shutting down.
